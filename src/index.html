Here are some of my projects:

Seru

Seru is a web application used for the tracking of games on sale from over 13 different online gaming distributors. Find the lowest price for games you want! This was a group project effort with 4 members built using HTML/CSS/Bootstrap/Javascript/Jquery/Spring/Java/Mysql

Coffee App

The Coffee app is a Javascript web application that allows users to sort, add, and delete a coffee by name or roast. This was a paired programming effort built using HTML/CSS/Javascript.

Weathermap App

Built a JavaScript application that gave a 5 day weather forecast for the area that the user searched or clicked on the map using both the Mapbox API and OpenWeatherApp API. This was a solo programming effort that was built using HTML/CSS/Javascript/jQuery.

Movie App

Built a JavaScript application that permitted users to edit, add, and delete movies from a webpage using jQuery. HTML/CSS
